Game Puzzle Wayang

The Puppet Puzzle Game is an educational game that aims to train players' intelligence and logic as well as being a means of learning about wayang culture. The main target of this game is children to teenagers (elementary school, middle school, high school/vocational school). In developing the Puzzle Wayang game, I played a role in determining the game idea, game analysis, UI/UX design, creating visual assets, developing the game's plot and mechanics.
This project is the result of individual PJBL (Project Base Learning).

Game Fruit Maze

The Fruit Maze game is a maze-themed game with fruit that challenges players to collect fruit by finding a way out. This game is equipped with various obstacles that increase the level of difficulty and time limits to add to the challenge. In developing the Fruit Maze game, I played a role in designing the game concept, designing UI/UX, creating visual assets, and determining the game flow and mechanics. The development of this game is only a mockup and has not been developed into a real game.
This project is the result of individual PJBL (Project Base Learning).
